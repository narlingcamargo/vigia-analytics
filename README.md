# vigia-analytics
1. Título y Descripción
Vigía Analytics: Análisis de Patrones de Inseguridad en Colombia (2020-2025)
Un estudio analítico enfocado en la identificación de patrones y tendencias de criminalidad, con especial énfasis en delitos de extorsión, utilizando herramientas avanzadas de ciencia de datos.

2. Tecnologías Utilizadas
Este proyecto integra un stack tecnológico diverso para el procesamiento y visualización de datos:

R: Limpieza de datos y análisis estadístico.

Python (Pandas/Matplotlib): Procesamiento de grandes volúmenes de datos y visualizaciones exploratorias.

Power BI: Creación de tableros (dashboards) interactivos para la toma de decisiones.

3. Metodología
Se procesaron datos históricos para identificar:

Patrones Temporales: Variaciones por mes, día y hora.

Análisis Geográfico: Identificación de zonas críticas (hotspots).

Comportamiento del Delito: Comparativa de modalidades de extorsión a lo largo del periodo 2020-2025.

4. Estructura del Repositorio
/BD ORIGINALES: Datos crudos tal como se obtuvieron.

/BD FINAL (JUNTAS): Bases de datos procesadas y unificadas.

/Scripts: Código fuente en R y Python.
